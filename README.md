# 🌿 Smart Crop Doctor  
### AI-Powered Plant Disease Detection System

🚀 A modern AI-based web application that detects plant diseases from leaf images and provides instant treatment suggestions.

---

## 🌟 Features

✅ AI-based plant disease detection  
✅ Supports multiple crops (Tomato, Potato, Pepper)  
✅ Instant prediction with confidence score  
✅ Treatment suggestions for each disease  
✅ Top 3 predictions visualization  
✅ Beautiful and modern UI  
✅ Fully deployable on cloud  

---

## 🧠 How It Works

1. User uploads a leaf image  
2. Image is preprocessed (resized & normalized)  
3. Deep Learning model predicts disease  
4. System displays:
   - Disease name  
   - Confidence score  
   - Treatment suggestion
